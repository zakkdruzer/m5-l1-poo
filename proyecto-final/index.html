Proyecto Final

Sistema de Gestión de una Librería

Integra clases, herencia, polimorfismo, encapsulamiento y JSON para administrar libros, préstamos, búsquedas y exportación/importación del catálogo.

Instrucciones

Qué debes construir

Implementa Libro, LibroDigital, Enciclopedia y Biblioteca. La biblioteca debe manejar una colección privada, préstamos, búsquedas, exportación a JSON e importación desde JSON.

El script debe probar agregados, validaciones, préstamos, devoluciones, búsquedas por autor, cálculo de lectura para libros digitales y generación de un reporte completo.

La lógica va en index.js. Abre la consola para ver el comportamiento del sistema.

Usa F12 para abrir la consola.
