Ejercicio 5

Figuras Geométricas Polimórficas

Crea una clase base Figura y varias subclases que sobreescriban area() para demostrar polimorfismo.

Instrucciones

Lo que debes implementar

Crea una clase base Figura con la propiedad color y un método area() abstracto que lance un error. Luego crea Triangulo, Rectangulo y Circulo usando extends y sobrescribe area().

Después crea un canvas con al menos 6 figuras, recórrelas con forEach, llama describir() en cada una y suma el área total.

La lógica va en index.js. Abre la consola para ver los resultados.

Usa F12 para abrir la consola.
